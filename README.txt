This is the README file for Training App


### Running the program ###
The program has been packaged as an executable application.
You can simply open up the executable "TrainingApp" in the "dist" folder.

### Creating a shortcut ###
For better accessibility, you may right click the executable mentioned above in
the "dist" folder, and then press "create shortcut". You can then place the shortcut
wherever is most convenient (desktop/other folder).
